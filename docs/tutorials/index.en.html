Step-by-step tutorials, in the spirit of MNE's: each pipeline stage is shown with the figure it produces, on one shared example subject.

Here is how to pass them.04Start from a T1, without DICOMYou already have a T1 as NIfTI (de-identified, or from another tool) and not the original DICOM.
