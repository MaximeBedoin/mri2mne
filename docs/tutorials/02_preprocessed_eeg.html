EEG déjà préprocessé

Votre EEG est déjà filtré et référencé (par ex. sorti d'un pipeline maison ou de MNE) ? Il faut empêcher le pipeline de le refaire.

Le principe

Refiltrer un signal déjà filtré déforme les basses fréquences et fausse la covariance ; re-référencer deux fois est une erreur silencieuse. On neutralise les deux étapes.

Les arguments

Mettre l_freq et h_freq à None désactive le filtrage ; eeg_reference=None conserve votre référence.

result = reconstruct_sources(
    subject="sample",
    output_dir="D:/derivatives",
    t1_path="D:/anat/sample_T1w.nii.gz",
    eeg_file="D:/eeg/sample_clean.fif",   # deja preprocesse
    digitization="D:/dig/sample.elc",
    simnibs_bin_dir="C:/Users/me/SimNIBS-4.5/bin",
    l_freq=None, h_freq=None,             # <- pas de refiltrage
    eeg_reference=None,                   # <- garder la reference
    events="find", event_id={"aud_l": 1},
)
Si vous avez seulement filtré (sans re-référencer), gardez eeg_reference="average". La référence moyenne commune, exprimée comme projection, est recommandée pour la modélisation de sources.

Le reste est identique au tutoriel complet : la réponse évoquée et les sources se calculent de la même façon.
